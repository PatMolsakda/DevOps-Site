# My DevOps Site

A simple DevOps-focused website created to demonstrate core DevOps concepts, tools, and workflows. This project can be used as a learning resource, personal portfolio, or starting point for practicing CI/CD and deployment.

## 📌 Purpose

* Practice and showcase DevOps fundamentals
* Demonstrate static website deployment
* Serve as a personal DevOps learning

## 🛠️ Technologies Used

* HTML5
* CSS3
* Git & GitHub
* GitHub Pages (for deployment)

## 📂 Project Structure

```
my-devops-site/           # The root of your project
├── .github/              # Hidden folder for GitHub configurations
│   └── workflows/        # Folder specifically for automation scripts
│       └── deploy.yml    # CI/CD pipeline logic (YAML)
├── index.html            # The main webpage
├── style.css             # The design file
└── README.md             # project documentation (for recruiters)
```

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/patmolsakda/my-devops-site.git
```

### 2️⃣ Open the Project

You can open the site locally by double-clicking `index.html` or using a live server extension in your code editor.

## 🌐 Deployment

This project can be deployed using **GitHub Pages**:

1. Go to **Repository Settings**
2. Navigate to **Pages**
3. Select branch: `main`
4. Select folder: `/root`
5. Save and access your live site
